# Phil Natividad - United Boxing Club Contact Page

## Files Included
- `index.html` - Main contact page
- `UBC_Logo_100_px.png` - UBC main logo (100px)

Both logos display in matching white circular frames.

## Important: File Structure
**All three files MUST be in the same directory for the logos to display correctly.**

## GitHub Pages Setup Instructions

### Option 1: Using GitHub Web Interface
1. Go to GitHub and create a new repository (e.g., "ubc-contact")
2. Click "uploading an existing file"
3. **Drag all three files together** into the upload area
4. Commit the files
5. Go to Settings â†’ Pages
6. Under "Source", select "main" branch and "/" (root)
7. Click Save
8. Your site will be live at: `https://[your-username].github.io/ubc-contact/`

### Option 2: Using Git Command Line
```bash
# Create a new directory and navigate to it
mkdir ubc-contact
cd ubc-contact

# Copy all three files to this directory
# (download them all to the same folder first)

# Initialize git repository
git init
git add .
git commit -m "Initial commit: UBC contact page"

# Create repository on GitHub, then:
git remote add origin https://github.com/[your-username]/ubc-contact.git
git branch -M main
git push -u origin main

# Enable GitHub Pages in repository settings
```

## Testing Locally
To test the page on your computer:
1. Download all three files
2. **Put them all in the same folder**
3. Double-click `index.html` to open in your browser

If logos don't appear, verify all three files are in the same directory.
